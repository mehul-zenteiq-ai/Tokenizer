# Byte-Level BPE (BBPE) Tokenizer Configuration

This repository contains the configuration for a high-capacity **Byte-Level Byte Pair Encoding** tokenizer. By operating at the byte level, this tokenizer eliminates "Unknown" (OOV) tokens and provides a robust foundation for multilingual language modeling.

---

## 📊 Specifications

| Feature | Details |
| :--- | :--- |
| **Model Type** | Byte-Level BPE (BBPE) |
| **Total Vocabulary** | 262,144 tokens |
| **Reserved Tokens** | 2,048 tokens |
| **Base Alphabet** | 256 (Raw UTF-8 Bytes) |
